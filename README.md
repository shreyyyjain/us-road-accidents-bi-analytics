# US Road Accidents Data Analytics (Power BI + BigQuery)

A cloud integrated business intelligence project analyzing accident severity, location-based hotspots, and environment-related risk patterns using BigQuery and Power BI.

## Objectives
- Identify severity and patterns across multi-year datasets
- Highlight high-risk locations and time trends
- Enable interactive drilldown analysis using BI dashboards

## Pipeline Summary
1. Data ingestion and storage in BigQuery
2. ETL transformations using Power Query
3. Dimensional modeling (facts and dimensions)
4. Dashboard built using Power BI

## Key Insights
- Severity distribution by conditions and location
- Weather and time dependent risk factors
- State and city-wise hotspot mapping
